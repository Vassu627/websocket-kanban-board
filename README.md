# WebSocket Kanban Board

Real-time Kanban board built with:

- React
- Node.js
- Socket.IO
- Vitest
- Playwright

### Backend

```bash
cd backend
npm install
node server.js
```

### frotend

```bash
cd frontend
npm install
npm run dev

```

### Test

```bash
npm run test

```

### E2E

```bash
npm run e2e
```
